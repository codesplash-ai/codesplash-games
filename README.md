# CodeSplash Games

CodeSplash Games is a desktop library of CodeSplash games for macOS, Windows, and Linux.

## Download

Download the latest version from the [CodeSplash Games releases page](https://github.com/codesplash-ai/codesplash-games/releases/latest).

The application checks this release feed for signed updates.

## Included games

- CodeSplash Snake
- CodeSplash Starship
- CodeSplash Chess
- CodeSplash City
- CodeSplash Horror
- CodeSplash Shooter

## Support and bug reports

Report problems or request features using [GitHub Issues](https://github.com/codesplash-ai/codesplash-games/issues).

## Source code

The application and game source repositories are privately maintained. This repository is used for public releases, documentation, and issue tracking.
